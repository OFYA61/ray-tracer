# Ray Tracer

Simple ray tracer built following [Ray Tracing in One Week](https://raytracing.github.io/books/RayTracingInOneWeekend.html) and [Ray Tracing: The Next Week](https://raytracing.github.io/books/RayTracingTheNextWeek.html#overview).

```bash
# Run the ray tracer to draw spheres
cargo run --release
```
